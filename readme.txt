Steps to run the code:

1. In the created ENV file enter the Access key and secret key for your aws account.
2. After entering the credentials run npm i to install the the dependencies
3. After running dependencies you ay either deploy the code to your account using sls deploy or in case if you want to run locally run command sls offline start
4. Note that in case of running locally you will need to create a dynamodb table named 'records' in your account.
